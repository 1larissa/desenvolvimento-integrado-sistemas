# Projeto-Desenvolvimento-Integrado-Sistemas

Projeto construído para a disciplina de Desenvolvimento Integrado de Sistemas da Universidade Tecnológica Federal do Paraná (UTFPR), com o objetivo de simular a comunicação entre sistemas em situações de alto consumo de memória e CPU.
<br>
Desenvolvido por Larissa Rodrigues e Camila Camila Miranda Marani.


Para rodar, é necessario ter o arquivo H-1.csv e H-2.csv dentro da pasta , não foram adicionados aos arquivos pelo tamanho!
